## Команды
    Установка зависимостей
    npm -i
    
    Запуск проекта в браузере
    npm start
    
    Сборка проекта в режиме разработчика
    npm run dev
    
    Сборка проекта для продакшена 
    npm run build
    
    Проверка стилий в проекте 
    npm run style
